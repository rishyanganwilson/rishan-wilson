# Rishan Wilson Landing Page

Static personal landing page for Rishan Wilson.

## Local preview

```powershell
python -m http.server 5500
```

Open `http://localhost:5500/`.

## GitHub Pages

This site can be deployed from the repository root with GitHub Pages:

1. Create a GitHub repository.
2. Upload `index.html`, `styles.css`, `script.js`, `.nojekyll`, and the `assets/` folder.
3. In GitHub, go to `Settings > Pages`.
4. Set source to `Deploy from a branch`.
5. Choose the `main` branch and `/root`.
